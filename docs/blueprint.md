# **App Name**: Cedat

## Core Features:

- Dynamic Content Presentation: Display a responsive and visually appealing layout for all static and dynamic content sections, including the hero, event cards, gallery grid, Startup World Cup timeline, and partner logos.
- Event Lifecycle Management: Store, retrieve, and display comprehensive event data (title, dates, location, images, description) via Firestore, supporting event registration.
- Community Gallery Showcase: Manage and present high-resolution community photos from a Firestore collection in an elegant, responsive masonry or justified grid with rounded corners.
- Ecosystem Metrics Display: Dynamically update and showcase key organizational statistics (members, startups, events, partners, projects) fetched from Firestore on the Home page.
- User Authentication & Registration: Securely manage user sign-up and login through Firebase Authentication, crucial for interactions like event registration and submitting inquiries.
- Interactive Form Submissions: Process user inputs from 'Register Now' buttons for events, storing relevant data in Firestore collections. The 'Ask Us' page will feature a simple, engaging UI with a title, description, and a button to redirect users to an external link for general inquiry submission for now.
- Startup World Cup Page Content Display: Display a dedicated page for the Startup World Cup featuring a vertical timeline of milestones (Applications Open, Close, Top 10, Regional Final, Grand Final) with teal circular icons and corresponding dates, a 'Why Attend?' section with three feature cards for Startups, Corporations, and Investors, and a grid showcasing global brand logos for partners.

## Style Guidelines:

- Primary color: Deep Teal (#008080) for a professional and tech-aligned feel. This color will highlight interactive elements and key headers.
- Background color: Ultra-light gray-teal (#ECF2F2), creating a clean and spacious canvas that subtly harmonizes with the primary color.
- Accent color: Vibrant Green (#3CC878), analogous to the primary hue, used for 'Upcoming' badges, circular icons in timelines, and to draw attention to call-to-action elements.
- Neutral colors: White (#FFFFFF) for card backgrounds and navigation bars, and Charcoal Gray (#313336) for body text and subdued elements to ensure high readability.
- Body and headline font: 'Inter' (sans-serif) for its modern, clean, and highly readable characteristics, suitable for all content types and a professional aesthetic.
- Leverage 'Lucide React' for a consistent and professional set of vector icons throughout the application, including those for dates, times, locations, and social media links.
- User Interface elements feature large rounded corners (rounded-2xl) and subtle card shadows, enhancing the modern aesthetic and visual depth.
- A sticky navigation bar with a white background and thin bottom border maintains brand visibility, while a dark-themed footer centralizes essential links and social presence.
- Subtle and smooth transitions on hover for interactive elements like cards and buttons, providing a polished and responsive user experience.
